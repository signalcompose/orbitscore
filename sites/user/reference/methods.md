# リファレンス（メソッド一覧）

このページは、困ったときに開いて該当箇所だけ参照するための早見表です。上から順番に読む必要はありません。

各セクションの見出しで目的のカテゴリに飛んでください。

---

## 1. グローバル設定

グローバルオブジェクト（慣例として `global` という変数名を使います）の設定です。すべての OrbitScore プログラムはここから始まります。

```text
var global = init GLOBAL
```

### グローバルメソッド一覧

| シグネチャ | 説明 | 例 |
|---|---|---|
| `tempo(N)` | テンポを BPM（1 分間に何拍鳴らすか）で設定する | `global.tempo(120)` |
| `beat(N by M)` | 拍子（タイムシグネチャ）を設定する。`beat(4)` のように M を省略すると `4` とみなす | `global.beat(4 by 4)` |
| `audioPath("path")` | `seq.audio()` で使う相対パスのベースディレクトリを設定する | `global.audioPath("./audio")` |
| `start()` | スケジューラーを起動する（シーケンス実行前に必須） | `global.start()` |
| `stop()` | すべてのシーケンスを停止する | `global.stop()` |

::: tip beat() の第 2 引数
`beat(N)` と `beat(N by M)` のどちらでも書けます。`beat(4)` は `beat(4 by 4)` と同じ意味です。
:::

## 2. シーケンス作成・基本設定

シーケンス（`seq`）はリズムや音のパターンを表す基本単位です。

```text
var kick = init global.seq
```

### 基本メソッド一覧

| シグネチャ | 説明 | 例 |
|---|---|---|
| `audio("file.wav")` | 再生するオーディオファイルを指定する。`audioPath` 設定時は相対パス可 | `kick.audio("kick.wav")` |
| `chop(N)` | オーディオファイルを N 個のスライスに均等分割する | `arp.chop(8)` |
| `play(...)` | スライス番号でリズムパターンを定義する（0 は休符） | `kick.play(1, 0, 1, 0)` |
| `length(N)` | ループの長さを N 小節に設定する（再生速度・音程が変わる） | `seq.length(2)` |
| `beat(N by M)` | このシーケンス固有の拍子を設定する（省略時はグローバルを継承） | `seq.beat(3 by 4)` |
| `tempo(N)` | このシーケンス固有のテンポを設定する（省略時はグローバルを継承） | `seq.tempo(90)` |

#### play() の数値の意味

- **0**: 休符（無音）
- **1〜N**: スライス番号（`chop(N)` で分割したスライスを指定）

```text
var arp = init global.seq
arp.audio("arpeggio.wav").chop(4)

// スライス 1→2→3→4 の順で再生
arp.play(1, 2, 3, 4)

// 休符を混ぜる
arp.play(1, 0, 3, 0)
```

#### ネストしたパターン

括弧 `()` でグループを作り、その拍を細分化できます。

```text
// 3 拍目を 2 分割する
kick.play(1, 0, (1, 0), 0)

// 4 拍目を 4 分割する
snare.play(0, 0, 1, (1, 1, 1, 1))
```

## 3. オーディオ操作

音量とステレオ位置を調整するメソッドです。

### オーディオ操作メソッド一覧

| シグネチャ | 説明 | 例 |
|---|---|---|
| `gain(dB)` | 音量を dB で設定する（0 がデフォルト、範囲: -60〜+12） | `kick.gain(-6)` |
| `pan(value)` | ステレオ位置を -100（左）〜100（右）で設定する（0 がデフォルト） | `hihat.pan(-50)` |
| `defaultGain(dB)` | 再生開始前に音量の初期値を設定する（再生トリガーなし） | `kick.defaultGain(-3)` |
| `defaultPan(value)` | 再生開始前にパンの初期値を設定する（再生トリガーなし） | `kick.defaultPan(-20)` |

::: info gain() と pan() は常に即時反映
`gain()` と `pan()` はアンダースコアの有無にかかわらず、常に即時反映されます。`gain(-6)` と `_gain(-6)` は同じ効果です。
:::

#### 音量の目安

| 値 | 効果 |
|---|---|
| `0` | 基準音量（デフォルト） |
| `-6` | 約半分の音量感 |
| `-12` | かなり小さい |
| `-60` | ほぼ無音 |
| `6` | 約 2 倍の音量感 |

#### パンの目安

| 値 | 位置 |
|---|---|
| `-100` | 完全に左 |
| `-50` | やや左 |
| `0` | 中央（デフォルト） |
| `50` | やや右 |
| `100` | 完全に右 |

## 4. アンダースコアプレフィックス（DSL v3.0）

OrbitScore v3.0 以降、ほぼすべてのメソッドに「即時適用版」（アンダースコアあり）があります。

### method() vs _method() の違い

| 書き方 | 動作 |
|---|---|
| `method(value)` | 設定を保存するだけ。次の `LOOP()` / `RUN()` が実行されたときに反映される |
| `_method(value)` | 設定を保存し、**即座に反映・再生開始する** |

ライブコーディング中に素早く変更を反映したいときは `_method()` を使います。

### 対応メソッド一覧

| 即時適用版 | 設定のみ版 | 対象 |
|---|---|---|
| `_audio("file.wav")` | `audio("file.wav")` | オーディオファイル指定 |
| `_chop(N)` | `chop(N)` | スライス分割 |
| `_play(...)` | `play(...)` | 再生パターン |
| `_beat(N by M)` | `beat(N by M)` | シーケンスの拍子 |
| `_length(N)` | `length(N)` | ループ長 |
| `_tempo(N)` | `tempo(N)` | シーケンスのテンポ |
| `_gain(dB)` | `gain(dB)` | 音量（どちらも即時反映） |
| `_pan(value)` | `pan(value)` | パン（どちらも即時反映） |

グローバルオブジェクトにも即時適用版があります。

| 即時適用版 | 効果 |
|---|---|
| `global._tempo(N)` | グローバルテンポを即時変更（継承しているシーケンスに反映） |
| `global._beat(N by M)` | グローバル拍子を即時変更（継承しているシーケンスに反映） |

::: tip 継承について
シーケンスは初期状態でグローバルのテンポ・拍子を継承します。`seq.tempo()` や `seq.beat()` を呼ぶと、その時点でそのシーケンスは独自の値を持ちます（グローバルの変更に追従しなくなります）。
:::

### 使用例

```text
// 設定フェーズ（再生前）: アンダースコアなしで書く
var global = init GLOBAL
global.tempo(120)
global.beat(4 by 4)
global.audioPath("./audio")

var kick = init global.seq
kick.audio("kick.wav")
kick.chop(1)
kick.play(1, 0, 1, 0)

global.start()

// 実行
LOOP(kick)

// --- ここから先は演奏中の書き換え ---

// パターンを即時変更（_play を使う）
kick._play(1, (1, 0), 1, 0)

// グローバルテンポを即時変更
global._tempo(140)
```

## 5. トランスポートコマンド（大文字キーワード）

シーケンスの再生・停止・ミュートを制御するコマンドです。すべて大文字で書きます。

### コマンド一覧

| コマンド | 説明 |
|---|---|
| `LOOP(a, b, …)` | 指定したシーケンスをループ再生する。指定しなかったシーケンスは自動停止 |
| `LOOP()` | すべてのループを停止する |
| `RUN(a, b, …)` | 指定したシーケンスを 1 回だけ再生する |
| `MUTE(a, b, …)` | 指定したシーケンスをミュートする（ループは継続、音だけ消える） |
| `MUTE()` | すべてのミュートを解除する |

### 使用例

```text
// kick だけループ
LOOP(kick)

// kick と snare をループ（他は停止）
LOOP(kick, snare)

// hihat を 1 回だけ再生
RUN(hihat)

// kick と snare をループしつつ hihat をミュート
LOOP(kick, snare, hihat)
MUTE(hihat)

// 全ミュート解除
MUTE()

// 全停止
LOOP()
```

::: warning LOOP は置き換え操作です
`LOOP(kick, snare)` を実行すると、以前から動いていた他のシーケンス（`hihat` など）は自動的に停止します。「追加する」のではなく「このリストに差し替える」という動作です。
:::

#### マルチライン記法

シーケンスが多い場合は改行して書けます。

```text
LOOP(
  kick,
  snare,
  hihat,
)

MUTE(
  hihat,
)
```

## 6. MIDI 出力（v2.0.0 Pitch DSL）

MIDI シーケンスと MIDI 関連のグローバル設定です。詳しくは [MIDI 出力](../midi/) を参照してください。

### グローバル — MIDI 設定

| シグネチャ | 説明 | 例 |
|---|---|---|
| `key("C")` | MIDI シーケンスのトニック（根音）を設定する ⚠️ | `global.key("C")` |
| `key("D3")` | 音名 + オクターブでトニックと基準レジスターを一括設定 ⚠️ | `global.key("D3")` |
| `midiLatency(ms)` | MIDI 送信の固定オフセット（SC とのタイミング合わせ、デフォルト 0） | `global.midiLatency(20)` |
| `quantize(val)` | LOOP 起動グリッド。`"bar"`(デフォルト) / `"beat"` / `"2bar"` / `"4bar"` / `"8bar"` / `"off"` | `global.quantize("bar")` |
| `linkAudio()` | LinkAudio モードを有効化（全オーディオ seq を Live へストリーム、macOS + Live 12.4+） | `global.linkAudio()` |
| `linkAudio(SR)` | SR（サンプルレート）を明示指定して LinkAudio を有効化 | `global.linkAudio(48000)` |

> ⚠️ `global.key()` / `seq.root()` などの root/key 系インターフェースは post-2.0 で設計見直し予定。

### シーケンス — MIDI 宣言

| シグネチャ | 説明 | 例 |
|---|---|---|
| `midi(port, ch)` | このシーケンスを MIDI シーケンスに切り替える（port は部分一致、ch は 1〜16） | `piano.midi("IAC", 1)` |
| `octave(N)` | 基準オクターブ（度数 1 の属するオクターブ、デフォルト 4 = C4） | `piano.octave(4)` |
| `vel(N)` | デフォルトベロシティ（1〜127、デフォルト 96） | `piano.vel(96)` |
| `gate(N)` | デフォルトゲート比（0.8 = スロットの 80%、デフォルト 0.8） | `piano.gate(0.8)` |
| `root(N)` | シーケンスデフォルトのルート（度数のみ）⚠️ | `piano.root(1)` |
| `voicelead()` | シーケンス全体に自動 voice lead を適用（エイリアス: `vl()`） | `piano.voicelead()` |
| `hold()` | 連続するスタック間の共通音を自動タイ | `piano.hold()` |

### シーケンス — MIDI play() の値

| 記法 | 意味 |
|---|---|
| `0` | 休符 |
| `1`〜`9`, `11`, `13` | 度数（Ionian 相対、根音からの音程） |
| `b3`, `#5` など | フラット / シャープ付き度数 |
| `5^+1` | 1 オクターブ上（スティッキー） |
| `5^-1` | 1 オクターブ下 |
| `1^0` | レジスターをリセット |
| `[1, 3, 5]` | コードスタック（同時発音） |
| `{1, 3, 5}` | レガートグループ（重なり発音） |
| `_` | タイ（直前の音を 1 スロット延長） |
| `5r` | 約 50% でランダムに鳴る（`Xr`） |
| `5^r` | 毎サイクル ±1 oct ランダム |
| `[1,3,5].r` | コードシニング（各声部 ~50% でランダム消灯） |
| `5@v110` | ベロシティ絶対値指定 |
| `5@v+20` | ベロシティ相対値（seq.vel() に加算） |
| `5@g30` | ゲートパーセント（30 = 0.30 = スタッカート） |
| `riff*3` | パターンを 3 スロット分繰り返す（`*n`） |

### シーケンス — コードとボイシング

| 記法 | 意味 |
|---|---|
| `var m7 = [1, b3, 5, b7]` | コード値（変数への代入） |
| `import chords` | 標準ライブラリ（`m7`, `maj7`, `dom7` など）を読み込む |
| `[m7, 9]` | スプレッド + 音の追加 |
| `[m7, -5]` | スプレッド + 音の削除 |
| `[1,3,5,7].drop(2)` | ドロップ 2 ボイシング |
| `[1,3,5,7].drop(2,4)` | ドロップ 2&4 |
| `[1,3,5,7].invert(2)` | 第 2 転回形（下 2 声を 1 oct 上げる） |
| `.open()` / `.close()` | オープン / クローズポジション |
| `.shell()` | R + 3 + 7 のシェルボイシング |
| `.rootless()` | ルートを除く |
| `([..], [..]).voicelead()` | グループ内コード間の声部進行を最小化 |

### シーケンス — モードとスコープ ⚠️

| 記法 | 意味 |
|---|---|
| `var dor = mode(1, 2, b3, 4, 5, 6, b7)` | モード（ピッチラティス）定義 |
| `(..).mode(dor)` | グループにモードを適用 |
| `(..).root(2)` | グループのルートを度数 2 に設定 |
| `(..).root(F)` | グループのルートをノート名 F に設定 |
| `(..).oct(1)` | グループを 1 oct 上げる（レキシカル、スコープ内のみ） |

### シーケンス — コンプ

| シグネチャ | 説明 | 例 |
|---|---|---|
| `comp(chord, ...)` | N コード → N 小節のコンプパターンに展開 | `piano.comp([1,3,5], [5,7,2])` |
| `cell("name")` | コンプセルを指定（`"charleston"` / `"quarters"` / `"twofour"` / `"redgarland"` / `"offbeats"`） | `piano.cell("quarters")` |
| `density(N)` | セルなし密度モード（0.0〜1.0） | `piano.density(0.6)` |

### シーケンス — LinkAudio

| シグネチャ | 説明 | 例 |
|---|---|---|
| `output("name")` | LinkAudio チャンネル名を指定（`global.linkAudio()` 宣言時必須） | `kick.output("kick")` |
| `quantize(val)` | このシーケンスの LOOP 起動グリッドを個別設定 | `fill.quantize("off")` |

## 7. メソッドチェーン

すべてのシーケンスメソッドは `.` で繋ぐことができます。

```text
// チェーン例
var drum = init global.seq
drum.audio("break.wav").chop(8).play(1, 3, 5, 7, 2, 4, 6, 8).gain(-6).pan(-20)

// 長い場合は改行してインデント
var arp = init global.seq
arp
  .audio("arpeggio.wav")
  .chop(8)
  .play(1, 2, 3, 4, 5, 6, 7, 8)
  .gain(-3)
  .pan(0)

// init 直後にチェーン
var snare = init global.seq
  .length(1)
  .audio("snare.wav")
  .chop(1)
  .play(0, 1, 0, 1)
  .gain(-3)
  .pan(20)
```

グローバルメソッドもチェーンできます。

```text
var global = init GLOBAL
global.tempo(120).beat(4 by 4)
global.start()
```
